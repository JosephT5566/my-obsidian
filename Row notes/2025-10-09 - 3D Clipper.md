# Description
是一個 google plugin
[Houzz Pro Clipper - Chrome Web Store](https://chromewebstore.google.com/detail/houzz-pro-clipper/lnhbnkmlkgeebjfimnbceakepeiandgg?pli=1)
可以幫助我們把 shophouzz （其他網站也ok） product 產品存到 Houzz pro 的 Library 中（My Products → In My Library）
# How to use it on staging
Clipper 本身是一個 iframe，因此我們可以在 dev tool 中，把原本的 pro.houzz 網址改成 pro.stghouzz，這樣就能讓 clipper 把 2D data info 存到 stghouzz 了
